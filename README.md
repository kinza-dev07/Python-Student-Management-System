# Student Management System

## Project Overview

This project is a simple Student Management System developed using Python. It allows users to store and manage student records through a menu-driven program. The project was created to practice Python programming concepts and improve logical thinking.

## Features

- Add a new student
- View all student records
- Search a student by roll number
- Update student information
- Delete a student record
- Easy to use
- Beginner-friendly

## How to Run

1. Install Python.
2. Download or clone this repository.
3. Open the project in Visual Studio Code.
4. Run the `student_management_system.py` file.
5. Follow the menu options displayed on the screen.
